# JARVIS - Voice Assistant

A native Python voice assistant built using CustomTkinter for the GUI. 
This is a college lab project.

## Team Members
- Sumit Kumar
- Tushar Poddar

## Features
- Personalization (Time-based Greetings & Identity handling)
- Knowledge (Wikipedia Summaries & Google Search via System Browser)
- Weather (OpenWeatherMap API integration)
- System Hardware Monitoring & Full-Screen Capture
- Power Control (OS-level Restart/Shutdown with voice confirmation)

## Setup
1. `pip install -r requirements.txt`
2. Run `python main.py`
3. Edit `src/features/weather.py` to add your OpenWeatherMap API key.
